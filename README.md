## 使用方法

### 本地预览

1. 直接在浏览器中打开 `index.html` 文件即可查看网站
2. 或者使用本地服务器：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js (需要安装http-server)
   npx http-server
   ```
3. 在浏览器中访问 `http://localhost:8000`

### 部署到服务器

1. 将所有文件上传到Web服务器
2. 确保服务器支持静态HTML文件
3. 访问域名即可查看网站

## 自定义修改

### 修改颜色主题
编辑 `css/style.css` 文件中的CSS变量：
```css
:root {
    --primary-color: #1e40af;    /* 主色调 */
    --secondary-color: #3b82f6;   /* 次要色调 */
    --accent-color: #f59e0b;      /* 强调色 */
}
```

### 修改联系方式
在各HTML文件中搜索 `0311-88855588` 进行替换

### 添加图片
如需添加公司Logo或产品图片，建议创建 `images/` 目录存放图片文件
